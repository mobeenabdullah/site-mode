Your all-in-one plugin for Maintenance Mode, Coming Soon, and Under Construction pages.

== Description ==

Welcome to Site Mode, the perfect plugin for easily switching your WordPress site into "Maintenance Mode" or creating exciting "Coming Soon" and "Under Construction" pages. Whether you're working on your site, giving it a makeover, or getting ready for a big launch, Site Mode is the must-have plugin for hassle-free website management.

**Easy to Use & Fully Customizable:**
Simply set up and customize your pages using Gutenberg blocks, making the process straightforward and user-friendly. Tailor your "Coming Soon" and "Maintenance" pages to reflect your brand with flexible options.

**Wizard Setup:**
Easily configure your pages with a wizard setup that lets you choose whether the page is "Coming Soon" or "Maintenance," and select templates for import.

**SEO-Friendly:**
Ensure optimal SEO with Site Mode's features, including customizable SEO meta titles, descriptions, and image settings.

**Integrations:**
Seamlessly integrate with tools like Google Analytics and Facebook Pixel to monitor and analyze user interactions and traffic.

**Advanced Settings:**
Fine-tune your website's behavior with advanced configuration options, including disabling the REST API, RSS feed, enabling redirects, whitelisting user roles, and pages.

**Live Demos:**
Check out Site Mode's collection of ready-made design templates. You can pick the one that best fits your style for both "Coming Soon" and "Maintenance Mode" pages. See the available templates at [https://site-mode.com](https://site-mode.com).

== Installation ==

When you activate the plugin, you'll be guided through a step-by-step setup wizard.

**Select Page Type:** Choose whether you want to create a "Coming Soon Page" to build anticipation or a "Maintenance Page" to inform users about site updates.
**Pick a Template:** Browse and select a design template that suits your website style.
**Customize with Ease:** Modify colors, show or hide elements like countdown timers, subscription forms, and social icons to match your preferences.
**Import Template:** Let the wizard import the necessary content and settings to create your page quickly.
**Preview and Edit:** After the wizard, you can preview or further customize your new page using Gutenberg blocks.

With this straightforward process, you can have your "Coming Soon" or "Maintenance Page" up and running in no time.

== Frequently Asked Questions ==

= What is Site Mode, and what does it do? =
Site Mode is a WordPress plugin that provides easy-to-use features for creating "Coming Soon" and "Maintenance Mode" pages for your website. The "Coming Soon" page allows you to build anticipation and excitement for your upcoming website launch, while the "Maintenance Mode" page informs visitors about temporary site unavailability during updates or maintenance.

= How do I install the Site Mode plugin on my WordPress website? =
Installing Site Mode is a simple process:
a. Download the Site Mode plugin from the WordPress repository.
b. Upload the plugin to your WordPress site.
c. Activate the plugin.
d. Follow the setup wizard to create your "Coming Soon" or "Maintenance Mode" page.

= Can I customize the appearance of my "Coming Soon" or "Maintenance Mode" page? =
Yes, you can fully customize the appearance of your page. You can choose from a variety of templates, adjust colors, and control which elements (such as countdown timers, subscription forms, and social icons) are displayed.

= Why the maintenance mode is not showing on my site? =
If the maintenance mode is not showing up on your site, there could be a few reasons. First, make sure you have enabled the plugin in the settings. Also, check that you are not signed in with a user role that is whitelisted, and that you have not whitelisted the page you are viewing.

= Is Site Mode SEO-friendly? =
Yes, Site Mode is designed to be SEO-friendly. You can set custom meta titles, meta descriptions, and images to ensure that your "Coming Soon" or "Maintenance Mode" page maintains good SEO practices.

= Are there any integrations available with Site Mode? =
Yes, Site Mode offers integrations with popular analytics tools like Google Analytics and Facebook Pixel, allowing you to monitor and analyze user interactions and traffic on your "Coming Soon" or "Maintenance Mode" page.

= Can I use Site Mode if I'm not a developer or have coding skills? =
Absolutely! Site Mode is designed for users of all technical levels. You can create your "Coming Soon" or "Maintenance Mode" page without any coding knowledge, thanks to its user-friendly interface and wizard-based setup.

= What happens after I've completed the setup wizard? =
After completing the setup wizard, you can preview your "Coming Soon" or "Maintenance Mode" page. If you wish to make further customizations, you can do so using the Gutenberg editor.

= Can I switch between "Coming Soon" and "Maintenance Mode" as needed? =
Yes, Site Mode allows you to easily switch between "Coming Soon" and "Maintenance Mode" as your website's needs change. You can toggle between these modes whenever you like.

= Is there a demo available to see Site Mode in action? =
Yes, you can explore Site Mode's features and see it in action on our demo site at https://site-mode.com

= Can I whitelist specific user roles or pages while in Maintenance Mode? =
Yes, Site Mode offers advanced configuration options, including the ability to whitelist user roles and specific pages. This ensures that certain users or pages can still access your site during maintenance, while others see the Maintenance Mode page.

= How can I disable certain features, such as the REST API or RSS feed, when my site is in Maintenance Mode? =
Site Mode provides advanced settings to easily disable features like the REST API and RSS feed during Maintenance Mode. This helps you control what information is accessible while your site is under maintenance.

= Is Site Mode compatible with the latest version of WordPress? =
Yes, Site Mode is designed to be fully compatible with the latest version of WordPress. We regularly update the plugin to ensure it works seamlessly with the most current WordPress releases.

= Is this plugin FREE? =
Yes, this plugin is Completely FREE
